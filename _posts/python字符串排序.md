# 题目
字符串排序。
# 题解
## 1.
### 1) 程序分析
### 2) 代码

```
# !/usr/bin/python3
# -*- coding: UTF-8 -*-


l = []
for i in range(3):
    l.append(input("input a string: "))
l.sort()
print(l)


```

### 3) 效果
```
input a string: a
input a string: ab
input a string: abc
['a', 'ab', 'abc']
```
