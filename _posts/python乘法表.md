# 题目
输出 9*9 乘法口诀表。

# 题解
## 1.
### 1) 程序分析
分行与列考虑，共9行9列，i控制行，j控制列。

### 2) 代码

```
# !/usr/bin/python3
# -*- coding: UTF-8 -*-


for i in range(1, 10):
    print()
    for j in range(1, i + 1):
        print('%d * %d = %d' % (i, j, i * j), end='  ')


```
**说明：** print 中默认 end='\n' 换行，换为 end='  '

## 2.
### 1) 程序分析
使用 format

### 2) 代码

```
# !/usr/bin/python3
# -*- coding: UTF-8 -*-


for i in range(1, 10):
    print()
    for j in range(1, i + 1):
        print('{} * {} = {}'.format(j, i, j * i), end='  ')

```
