# 题目
循环输出列表
# 题解
## 1.
### 1) 程序分析
### 2) 代码

```
# !/usr/bin/python3
# -*- coding: UTF-8 -*-


s = ["man", "woman", "girl", "boy", "sister"]
for i in range(len(s)):
    print(s[i])

```

### 3) 效果
```
man
woman
girl
boy
sister
```
