# 题目
创建一个链表。

# 题解
## 1.
### 1) 程序分析
### 2) 代码

```
# !/usr/bin/python3
# -*- coding: UTF-8 -*-

l = []
for i in range(5):
    l.append(int(input('input a number: ')))
print(l)

```

### 3) 效果
```
input a number: 1
input a number: 2
input a number: 3
input a number: 4
input a number: 5
[1, 2, 3, 4, 5]
```
